# Python Security Tools

This repository contains beginner-friendly Python projects related to cybersecurity, digital forensics, and security automation.

## Included Tools

### Hash Calculator

Generates MD5, SHA1, and SHA256 hashes from user-provided text.

### File Hash Verifier

Calculates the SHA256 hash of a file to support integrity verification and digital evidence validation.

### Log Analyzer

Searches log files for specific keywords and displays matching entries.

### Port Scanner

Performs basic TCP port scanning against a target host.

### Password Generator

Generates random passwords using letters, numbers, and special characters.
